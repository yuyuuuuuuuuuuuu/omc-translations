How many 999-digit positive integers satisfy all the following conditions?

  • The digits 111 through 999 are each used exactly once.
  • When we extract only the digits 2,3,4,52, 3, 4, 52,3,4,5, they appear in this order from left to right.
  • When we extract only the digits 6,7,8,96, 7, 8, 96,7,8,9, they appear in this order from left to right.

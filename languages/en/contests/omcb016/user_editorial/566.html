Problems of this kind often allow us to express all polynomials based on one polynomial that satisfies

$\tag{1} P(3)=64, P(8)=9, P(13)=4$
.
Therefore, the first thing we should do is to create at least one polynomial that satisfies equation (1)(1)(1).
One way to do this is to set f(x)=ax2+bx+cf(x)=ax^2+bx+cf(x)=ax2+bx+c and then solve
f(3)=64,f(8)=9,f(13)=4f(3)=64, f(8)=9, f(13)=4f(3)=64,f(8)=9,f(13)=4
, which is a valid method.

However, let's consider if we can be more creative here. The numbers 64,9,464, 9, 464,9,4 are all perfect squares. Noticing this might make us want to take square roots. If we take the positive square roots, we get

g(3)=8,g(8)=3,g(13)=2g(3)=8, g(8)=3, g(13)=2g(3)=8,g(8)=3,g(13)=2
, which is not particularly useful. However, if we realize that we can instead set g(13)=−2g(13)=-2g(13)=−2, we find
g(3)=8,g(8)=3,g(13)=−2g(3)=8, g(8)=3, g(13)=-2g(3)=8,g(8)=3,g(13)=−2
, and we can see that g(x)=11−xg(x)=11-xg(x)=11−x is one solution. This means that f(x)=(11−x)2=(x−11)2f(x)=(11-x)^2=(x-11)^2f(x)=(11−x)2=(x−11)2 is one solution to equation (1)(1)(1), and we can express it as
P(x)=Q(x)(x−3)(x−8)(x−13)+(x−11)2P(x)=Q(x)(x-3)(x-8)(x-13)+(x-11)^2P(x)=Q(x)(x−3)(x−8)(x−13)+(x−11)2
.

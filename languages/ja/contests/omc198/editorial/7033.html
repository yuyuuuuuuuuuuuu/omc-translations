結論から述べれば，ある状態から始めて後手が勝つことは以下と同値である：

  • 各山の石の数を二進法で表したとき，どの位についてもそれが 111 であるものが 333 の倍数個存在する．

この状態を以下では良い状態とよび，そうでない状態を良くない状態とよぶ．
　最後が良い状態であることに注意すれば，以下の二つを示せばよい：

  • どの良い状態からどのように 111 回操作しても良くない状態の盤面になること．
  • どの良くない状態からも，適当に 111 回操作することで良い状態の盤面にできること．

以下，特に断りなく「位」などといったとき，二進法による表記で考えるものとする．
　まず前者から示す．どの位についても，それが 111 であるような山の数の差は， 111 回の操作の前後で 222 以下である．よって，良い状態から操作して再び良い状態であるためには，どの位についても操作前と後でその位が 111 であるような山の数が変わらない必要がある．これは操作の前後で石の総数が変わらないことを意味するので不適である．
　以下，後者を示す．各位についてその位が 111 であるような山の数を 333 で割った余りを考える：

  • 000 でない最上位の位の余りの数が 222 の場合に行うべき操作：
    　その位が 111 であるような 222 つの石の山を選び，その位がそれぞれ 000 になるようにする．
    　それより下の位についても，同様にして調整が可能である．

  • 000 でない最上位の位の余りの数が 111 の場合に行うべき操作：
    　まず，その位が 111 であるような 111 つの石の山を選び，その位を 000 になるようにする．
    　それより下の位について，先に選択した以外の山であってその位が 111 であるようなもの数を 333 で割った余りを考える．このとき，余りが 111 である位がなければ，先に選択した石の山での調整のみですべての位の余りを 000 にできる．そうでない場合も，余りが 111 である位の最上位が 111 であるような石の山を新たに 222 つ目として選択すれば，これらを利用して調整が可能である．

1. Reasons for writing the user editorial

The official explanation is very concise and elegant, but I thought it might be difficult for those who cannot solve the problem to understand the explanation, so I tried to write it in as simple a way as possible.

  1. Explanation of ppnp_{p_n}ppn​​

The sequence {pn}\{ p_n \}{pn​} can be seen as a sequence at first glance, but it can also be viewed as a function.
That is, it is a function that maps elements of the set S={1,2,3,⋯ ,13}S=\{1,2,3, \cdots, 13\}S={1,2,3,⋯,13} to elements of SSS (to be more precise, it is better to say it is a bijective mapping).
So what is ppnp_{p_n}ppn​​?
For a number nnn, it is the result of applying the function ppp once, and then applying the function ppp again. (As shown below)

n→ppn→pppnn \xrightarrow{p} p_n \xrightarrow{p} p_{p_n}np​pn​p​ppn​​
In the following explanation, there are instances where applying the function ppp twice is represented as p2p^2p2. That is,

  1. Reasons for considering cycles

For a number nnn, consider repeating the function ppp. That is, it forms a sequence like the following:

n→ppn→pppn→p⋯n \xrightarrow{p} p_n \xrightarrow{p} p_{p_n} \xrightarrow{p} \cdotsnp​pn​p​ppn​​p​⋯
The values transformed by ppp become elements of the set SSS, and since the elements of set SSS are finite, this sequence will eventually have the same number appear twice. Let that number be aaa, then
a→ppa→p⋯→paa \xrightarrow{p} p_a \xrightarrow{p} \cdots \xrightarrow{p} aap​pa​p​⋯p​a
This shows that a cycle exists due to ppp.

  1. Considering examples of cycles

First, let's consider the most basic cycle of length 111.

In this case, ppa=ap_{p_a}=appa​​=a, which contradicts the conditions of the problem. Similarly, a cycle of length 222 also leads to a contradiction (try to verify this).
Next, consider a cycle of length 333.
a→ppa→pppa→paa \xrightarrow{p} p_a \xrightarrow{p} p_{p_a} \xrightarrow{p} aap​pa​p​ppa​​p​a
Here, if ppn≤n−1p_{p_n} \leq n-1ppn​​≤n−1 holds for all nnn, it clearly leads to a contradiction (since a>ppa>pa>aa \gt p_{p_a} \gt p_a \gt aa>ppa​​>pa​>a). Therefore, there must be at least one case where ppn=n+1p_{p_n} = n+1ppn​​=n+1. Let's redefine it as

From this, we can conclude that pap_apa​ is either a+2a+2a+2 or a−1a-1a−1. More generally, if there is a cycle of length 333, the elements contained within it are three consecutive integers.

Next, consider a cycle of length 444. For simplicity,

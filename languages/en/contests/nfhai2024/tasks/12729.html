A string where each character is either uuu, nnn, or ooo is called a good string. For good strings, we define an operation as follows:

  • Operation: Choose two adjacent characters, swap their positions, and then rotate both characters 180∘180^\circ180∘ (it can also be considered as rotating the two characters as a single unit 180∘180^\circ180∘).

Here, it is assumed that through 180∘180^\circ180∘ rotation, nnn changes to uuu, ooo remains ooo, and uuu changes to nnn. For example, if an operation is performed on nonono, it becomes ououou. In this case, please answer the number of good strings that can be transformed into

nnunounnunounnunou
after a finite number of operations.

We need to find the number of positive integers that satisfy the following conditions.

  • (a) It has exactly 444 digits in decimal notation.
  • (b1) It is a multiple of 333.
  • (b2) The digit in the units place is even.
  • (c) The digits in decimal notation consist of exactly 222 different types.

If a positive integer meets (a), (b1), and (c), then the digits can be classified into the following categories.

  • (i) All are multiples of 333.
  • (ii) One digit is a multiple of 333 and three digits are not divisible by 333.
  • (iii) Two digits leave a remainder of 111 when divided by 333, and two digits leave a remainder of 222.

For each category, those that also satisfy (b2) can be counted as follows.

  • (i): When the units digit is 000, there are 3×22=123\times 2^2=123×22=12 possible numbers, and when it is 666, there are 2×(23−1)+3=172\times (2^3-1)+3=172×(23−1)+3=17 possible numbers.
  • (ii): When the units digit is a multiple of 333, there are 2×6=122\times 6=122×6=12 possible numbers, and when it is not, there are 3×(3×3+2)=333\times(3\times 3+2)=333×(3×3+2)=33 possible numbers.
  • (iii): Considering the units digit, there are 3×3×3=273\times 3\times 3=273×3×3=27 possible numbers.

Therefore, the total number of integers satisfying the conditions is 101\mathbf{101}101.

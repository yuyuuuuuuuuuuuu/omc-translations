素数 p=109+7p = 10^9 + 7p=109+7 について，N=4p+7N = 4p + 7N=4p+7 とします．
　NNN 個の都市からなる国があり，はじめどの 222 つの都市の間にも橋がかかっていません．NNN 個の都市にはそれぞれ 111 匹ずつ猫がおり，各猫は次のような移動を同時に開始します．

  • 今いる都市から橋をちょうど 111 本渡って行くことができる都市を 111 つ選び，その都市にちょうど 111 時間かけて移動する (途中で引き返すことはしない)．

ここで，111 つの橋の上で複数の猫がすれ違うこと，および 111 つの都市に複数の猫がいることはできないものとします．このとき，都市間に合計 N+1N+1N+1 本の橋をかける方法であって，次の条件をみたすものが MMM 通りあるとします．

  • いずれの橋も相異なる 222 つの都市の間にかかっており，どの相異なる 222 つの都市の間にも，橋は高々 111 本しかかかっていない．
  • どの都市にも少なくとも 111 本は橋がかかっている．
  • 移動を NNN 時間連続で行うことができる．

　このとき，MMM を素数 ppp で割った余りを解答して下さい．

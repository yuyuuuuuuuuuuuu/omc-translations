Let nnn be an integer between 222 and 200200200. A number that can be written in base nnn with exactly n−1n-1n−1 digits, where each digit is a permutation of 1,2,…,n−11, 2, \ldots, n-11,2,…,n−1, is called a good number in base nnn. For example, 1234(5)1234_{(5)}1234(5)​ and 2431(5)2431_{(5)}2431(5)​ are good numbers in base 555, but 12340(5)12340_{(5)}12340(5)​ and 3141(5)3141_{(5)}3141(5)​ are not good numbers in base 555.

In this case, please find the sum of all positive integers nnn that satisfy the following condition:

  • Condition: All good numbers in base nnn are divisible by n−1n-1n−1.

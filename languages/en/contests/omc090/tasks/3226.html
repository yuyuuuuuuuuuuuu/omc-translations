We will color the vertices of a regular polygon with nnn sides satisfying the following conditions:

  • Use only the 101010 colors: black, gray, tan, green, cyan, blue, yellow, orange, red, and purple, with each color used at least once.
  • For all diagonals, the colors of the 222 endpoints must be different.

Find the sum of all integers n≥10n \geq 10n≥10 for which this is possible.

黒く塗りつぶされるマスの条件は

  1. a, ba,\, ba,b の少なくとも一方が 101010 の倍数，
  2. a, ba,\, ba,b の少なくとも一方が 111，
  3. a, ba,\, ba,b がともに 111 桁，
  4. a>ba \gt ba>b

であるから，これらを一つずつ適用することを考える．ただし，左上から右下への対角線上にあるマスを単に「対角線上にある」と表現することとし，対角線上にあるマスとそうでないマスを分けて考える．
　まず 1. と 2. を適用する．222 桁以下の正の整数のうち，101010 の倍数でも 111 でもないものは 99−9−1=8999 - 9 - 1 = 8999−9−1=89 個あるので，1. と 2. により塗りつぶされずに残るマスは 89289^2892 個存在する．このうち対角線上にあるものは 898989 個ある．
　次に 3. を適用する．111 桁の 111 より大きい整数は 888 個あり，いずれも 101010 の倍数でないので，3. により塗りつぶされず残るマスは 892−82=81⋅9789^2 - 8^2 = 81 \cdot 97892−82=81⋅97 個存在する．このうち対角線上にあるものは 89−8=8189 - 8 = 8189−8=81 個ある．
　最後に 4. を適用する．対角線上にないマスのうち半分が塗りつぶされるので，最終的に塗りつぶされずに残るマスの数は 81⋅97−812+81=81⋅49=3969 \frac{81 \cdot 97 - 81}2 + 81 = 81 \cdot 49 = \bm{3969}281⋅97−81​+81=81⋅49=3969 である．

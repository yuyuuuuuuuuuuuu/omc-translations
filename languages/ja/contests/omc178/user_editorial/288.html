実は BD=CDBD = CDBD=CD は様々な方法で示すことができます. (BD=CDBD = CDBD=CD を示した後の方針は公式解説と同様です.) そこで, ここでは BD=CDBD = CDBD=CD を示す 222 つの方法の大まかな流れを紹介します. \\ 方針1

  • 三角形 BECBECBEC の外心を FFF とする. すると, 三角形 FECFECFEC は正三角形となる.
  • ∠FCD=∠ECD=30∘∠FCD = ∠ECD = 30^\circ∠FCD=∠ECD=30∘ より, 三角形 FCDFCDFCD と三角形 ECDECDECD は合同と分かり, ∠FDC=∠EDC=42∘∠FDC = ∠EDC = 42^\circ∠FDC=∠EDC=42∘ と分かる.
  • DF⊥BCDF \perp BCDF⊥BC, 及び BF=CFBF = CFBF=CF から直線 DFDFDF が BCBCBC の垂直二等分線と分かり, BD=CDBD = CDBD=CD が従う.

方針2

  • 三角形 DECDECDEC の外心を LLL とする. すると, 三角形 DLEDLEDLE は正三角形となり, 簡単な角度計算により ∠BCL=30∘∠BCL = 30^\circ∠BCL=30∘ が分かる.
  • 直線 CLCLCL と直線 BEBEBE の交点を KKK とすると, BK=CKBK = CKBK=CK , 及び ∠LKE=60∘∠LKE = 60^\circ∠LKE=60∘ が分かる.
  • ∠LKE=∠LDE=60∘∠LKE = ∠LDE = 60^\circ∠LKE=∠LDE=60∘ より, 四点 K,D,E,LK, D, E, LK,D,E,L が共円と分かり, 円周角の定理より ∠KDL=∠KEL=24∘∠KDL = ∠KEL = 24^\circ∠KDL=∠KEL=24∘ が分かる.
  • ∠KDC=∠KDL+∠LDC=42∘∠KDC = ∠KDL + ∠LDC = 42^\circ∠KDC=∠KDL+∠LDC=42∘ より DK⊥BCDK \perp BCDK⊥BC が, BK=CKBK = CKBK=CK より直線 DKDKDK が BCBCBC の垂直二等分線であることが従い, BD=CDBD = CDBD=CD が従う.

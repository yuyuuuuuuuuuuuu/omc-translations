  • Initially, choose 222 distinct vertices and mark them.
  • Select an unmarked vertex XXX and two distinct marked vertices A,BA,BA,B, and if all of the following hold then mark XXX; repeat this operation until there are no vertices XXX that can be marked.
    • XA=XBXA=XBXA=XB holds.
    • For any vertex CCC other than the marked A,BA,BA,B, the segment ABABAB and the segment CXCXCX have a common point. If the number of marked vertices is only 222, this condition is regarded as satisfied.

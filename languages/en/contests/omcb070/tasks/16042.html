There are 222 pieces placed in the left bottom corner and the right top corner of a 5×55×55×5 grid. In one operation, you can perform one of the following 333 actions:

  • Select one piece on the grid and move it to an adjacent square. However, only one piece can be placed in a single square.
  • Select a pair of adjacent squares that both contain pieces and remove the 222 pieces from those 222 squares.
  • Select a pair of adjacent squares that are both empty and place 111 new piece in each of those 222 squares.

What is the number of ways to remove all pieces from the 5×55×55×5 grid with the minimum number of operations? Note that "adjacent 222 squares" refers to 222 squares that share an edge on the grid.

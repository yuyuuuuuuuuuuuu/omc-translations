There is a grid of 999×999999×999999×999. We define a way to write one integer between 111 and 999999999 in each of several cells as shining writing if it satisfies the following conditions:

  • In each row and each column, exactly 111 number is written.
  • For i=1,2,…,999i=1,2,\dots,999i=1,2,…,999, the number iii is written in either the iiith row or the iiith column.

Let nnn be the number of integers iii such that iii is written in the cell at the iiith row and iiith column in a shining writing. We define the brightness of this writing as 2n2^n2n. We are required to find the maximum number of times the total sum of the brightness of all shining writings can be divided by 555.

Taro, the dice craftsman, received an order for a special die that satisfies the following conditions:

  • It is a regular hexagonal die.
  • Each face must show an integer between 111 and 666.
  • It is allowed to have multiple faces showing the same number.
  • For any edge, the sum of the numbers on the two faces sharing that edge must be a prime number.

How many ways are there to create a die that satisfies these conditions? Note that the orientation of the numbers on each face is not considered, and configurations that can be obtained by rotating the die to make all faces show the same arrangement are not distinguished.

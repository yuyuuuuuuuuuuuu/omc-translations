「 QQQ は FGFGFG に関して AAA と対称な点である」までは公式解説と同様です．
　Brocard の定理より，円 ABCABCABC において，FGFGFG は極を EEE とする極線です．従って，EEE から FGFGFG に下ろした垂線の足を HHH とすると，O,E,HO,E,HO,E,H は一直線上にあり，OE×OH=302OE \times OH = 30^2OE×OH=302 が成立します．一方で OE2OE^2OE2 は方べきの定理から求められますから，EH=OH−OEEH = OH-OEEH=OH−OE の値もこの時点で計算することができます．
　一方で，直線 CACACA と FGFGFG の交点を RRR とすると，完全四辺形の性質から C,E,A,RC,E,A,RC,E,A,R は調和点列ですから，ARARAR の長さが求められます．
　以上より，直線 FGFGFG と C,E,AC,E,AC,E,A それぞれとの距離（従って QQQ との距離も）を計算することが可能になりました．あとは愚直に三平方の定理を用いて計算することで CQCQCQ の長さを求めることができます．

　こういう「三平方の定理でゴリ押せないか？」という思考は、エレガントではないかもしれませんが、OMC で役に立つことは多いです。

In a 4×44\times 44×4 grid, a set PPP consisting of one or more cells is called a puzzle if it satisfies the following conditions:

  • Between any two distinct cells of PPP, you can move by following several adjacent cells that belong to PPP. However, a set consisting of only one cell is considered a puzzle.

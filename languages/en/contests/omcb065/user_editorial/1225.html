For the top left cell, a coin must be placed either in itself or in one of the adjacent cells, and the same applies to the top right, bottom left, and bottom right, so at least 444 coins are needed.
The reason why it is not possible to place 999 or more coins is the same as explained in the official explanation.
By coloring the cells in a checkerboard pattern and placing coins on one color, it is possible to place 888 coins. When coins are placed in the cells "Row 1, Column 2", "Row 2, Column 4", "Row 3, Column 1", and "Row 4, Column 3", it is clear that no more coins can be placed, so the maximum value sought is 888 and the minimum value is 444.

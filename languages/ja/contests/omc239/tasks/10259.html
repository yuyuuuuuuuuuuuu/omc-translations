111 から 3113^{11}311 までの整数が書かれたカードがそれぞれ 111 枚ずつあり，左から小さい順に横一列に並んでいます．111 枚以上のカードが重なった状態をカード束と呼びます．
　3n3n3n 個のカード束が横一列に並んでいるとき，カード束を左から順に X1,…,X3nX_1,\ldots,X_{3n}X1​,…,X3n​ とし，以下の 333 種類の操作のうち 111 つを行うことができます．

  • 操作 AAA：111 以上 nnn 以下の任意の整数 kkk に対して，Xn+kX_{n+k}Xn+k​ を X2n+kX_{2n+k}X2n+k​ の上に重ね，さらにその上に XkX_{k}Xk​ を重ねることで nnn 個のカード束を得る．
  • 操作 BBB：111 以上 nnn 以下の任意の整数 kkk に対して，X2n+kX_{2n+k}X2n+k​ を XkX_{k}Xk​ の上に重ね，さらにその上に Xn+kX_{n+k}Xn+k​ を重ねることで nnn 個のカード束を得る．
  • 操作 CCC：111 以上 nnn 以下の任意の整数 kkk に対して，XkX_{k}Xk​ を Xn+kX_{n+k}Xn+k​ の上に重ね，さらにその上に X2n+kX_{2n+k}X2n+k​ を重ねることで nnn 個のカード束を得る．

　このとき，操作 A,B,CA,B,CA,B,C を合計 111111 回行うことでカード束がちょうど 111 つになります．すべての操作が終了した後のカード束の上から kkk 枚目に書かれている整数を aka_kak​ とおくとき，aam=1a_{a_m}=1aam​​=1 を満たす整数 mmm がちょうど 111 つ存在するため，この mmm をその操作のスコアとします．
　666 回目の操作で操作 AAA を行うような，111111 回の操作の実行方法は 3103^{10}310 通りありますが，これらすべてに対するスコアの総和を求めてください．

例 　例えば，1,2,3,4,5,6,7,8,91,2,3,4,5,6,7,8,91,2,3,4,5,6,7,8,9 という 999 枚のカード (束) が横一列に並んでいるときに，操作 AAA の後に操作 BBB を行うと，カード束は 111 個になり，上から 2,5,8,3,6,9,1,4,72,5,8,3,6,9,1,4,72,5,8,3,6,9,1,4,7 の順に並びます．

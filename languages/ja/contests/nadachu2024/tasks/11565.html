作問：山口

　以下の 　　\fbox{　　}　　​ に当てはまる整数を解答してください．

　mmm と nnn を 666 以上の整数とする．正五角形 ABCDEABCDEABCDE の辺 ABABAB を一辺とする正 mmm 角形，そして正 nnn 角形を図のように書き，正 mmm 角形上に点 F,GF,GF,G を，正 nnn 角形上に点 H,IH,IH,I をそれぞれ図のように定める（m,nm,nm,n の間には大小の制約を定めない）．そして，CHCHCH を一辺とする正 mmm 角形上に点 PPP を，EGEGEG を一辺とする正 nnn 角形上に点 QQQ を図のようにそれぞれとる．このとき，PPP と QQQ が一致するような mmm と nnn の組は 　　\fbox{　　}　　​ 通りある．

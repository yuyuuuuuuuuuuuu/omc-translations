1. はじめに

　この解説は公式解説の解説のような立ち位置です.
そのため解説としての新規性は特にありません.

2. 最適な交換手順その 111

　まず以下の点に留意します.

  • 交換できる空容器があれば交換した方が良い.
  • 交換の順番は結果に影響しない.

すると, 例えば以下のような交換の手順その 111 が思いつきます.
(ただし, 全て飲料は得た瞬間に飲み干しているとします.)

  • AAA を全て飲み, AAA の空容器を交換できるだけ BBB に交換する.
  • BBB を全て飲み, BBB の空容器を交換できるだけ AAA に交換する.
  • 上記の 222 つを交換できる空容器がある限り繰り返す.

最初に見た留意点より, この手順は最適な交換手順(の一例)となります.

交換手順1の具体例

具体的に, A,BA,BA,B を 401401401 本ずつ持っているときの容器の数の挙動を見てみましょう.
(a,b)(a,b)(a,b) と書いたら A,BA,BA,B の容器をそれぞれ a,ba,ba,b 個持っている状態を表すとします.

  1. (401,401)(401,401)(401,401) から, AAA の空容器を 399399399 個交換して, BBB を 133133133 個得る. (2,534)(2,534)(2,534) となる.
  2. (2,534)(2,534)(2,534) から, BBB の空容器を 532532532 個交換して, AAA を 767676 個得る. (78,2)(78,2)(78,2) となる.
  3. (78,2)(78,2)(78,2) から, AAA の空容器を 787878 個交換して, BBB を 262626 個得る. (0,28)(0,28)(0,28) となる.
  4. (0,28)(0,28)(0,28) から, BBB の空容器を 282828 個交換して, AAA を 444 個得る. (4,0)(4,0)(4,0) となる.
  5. (4,0)(4,0)(4,0) から, AAA の空容器を 333 個交換して, BBB を 111 個得る. (1,1)(1,1)(1,1) となる.
  6. 交換できるものが無いため終了する.

この手順自体はかなり直感的だと思いますが, 各操作で, 一方の種類の容器が減りもう一方が増えるため, 容器数の挙動が追いにくいという欠点があります.
そこで, 次はこれを改善してみましょう.

3. 最適な交換手順その 222

　第二節で述べた問題点について考えます.
　ここでの問題点とは, AAA を減らしたら BBB が増えることにあるのでした.
そこで, AAA を BBB に交換した直後に, 増えた分の BBB を全て AAA に交換したら BBB の本数を変えることなく, AAA の本数の変化が実現できるのでは, という発想に至ります. ( B→A→BB\to A\to BB→A→B の交換も同様に考えられます. )

　このアイデアを解法に落とし込むために細部を詰めていきましょう.

まず, 先ほどはある分の AAA を全てまとめて BBB に交換していましたが, すると剰余の関係で, 増えた分の BBB を全て AAA に交換できるとは限りません.
これの解決法としては, 先ほどから使っている "交換の最適性は手順に依らない" という事実が使えます.
具体的には, 全てまとめて交換するのではなく, 交換で増える BBB が全て AAA に変換できる数ずつ, さらに具体的には 3×7=213\times 7=213×7=21 本ずつ AAA を BBB に交換をすれば良いです.

次に, この方法で最後まで交換しきれるかが非自明であるという問題点があります.
しかし, これはこの段階では気にしなくて良いです.
というのも, もし最後に一部の空容器が余ったとしても, それは高々 202020 本以下ずつであるので, その後手順その 111 をすれば十分手計算できそうだからです.

以上をまとめると, 次のような交換の手順その 222 を得ます.

  • 飲んでいない AAA が 212121 本以上あれば, AAA 212121 本を飲み, 新規の BBB 777 本に交換し, 交換で得た BBB 777 本を全て飲んで新規の AAA 111 本に交換する.
  • 飲んでいない BBB が 212121 本以上あれば, BBB 212121 本を飲み, 新規の AAA 333 本に交換し, 交換で得た AAA 333 本を全て飲んで新規の BBB 111 本に交換する.
  • 上記のを少なくとも一方ができる間繰り返す.
  • 繰り返し終了時にもし残りがあれば手順その1のようにする.

交換手順2の具体例

具体的に, A,BA,BA,B を 401401401 本ずつ持っているときの容器の数の挙動を見てみましょう.
(a,b)(a,b)(a,b) と書いたら A,BA,BA,B の容器をそれぞれ a,ba,ba,b 個持っている状態を表すとします.

  1. (401,401)(401,401)(401,401) から, AAA 212121 個を交換して, BBB を 777 個得る. (380,401+7)(380,401+7)(380,401+7) となる. 得た 777 個の BBB を, 111 個の AAA に交換する. (381,401)(381,401)(381,401) となる.
  2. (381,401)(381,401)(381,401) から, AAA 212121 個を交換して, BBB を 777 個得る. (360,401+7)(360,401+7)(360,401+7) となる. 得た 777 個の BBB を, 111 個の AAA に交換する. (361,401)(361,401)(361,401) となる.
  3. これを続ける.

この各手順のでの前後の本数や, その間に飲んだ飲料数をみれば, これは以下と同じであると分かります.

  • 飲んでいない AAA が 212121 本以上あれば, AAA 212121 本を飲み, 新規の AAA 111 本に交換し, それとは別に BBB 777 本を飲んだという事実のみを得る.
  • 飲んでいない BBB が 212121 本以上あれば, BBB 212121 本を飲み, 新規の BBB 111 本に交換し, それとは別に AAA 333 本を飲んだという事実のみを得る.
  • 上記のを少なくとも一方ができる間繰り返す.
  • 繰り返し終了時にもし残りがあれば手順その1のようにする.

さらに, 飲料を飲み交換するところを, 飲んだ事実を得るパートと交換するパートに分けて考えることで,

  • 飲んでいない AAA が 212121 本以上あれば, 飲んでいない AAA 202020 本を減らし, それとは別に BBB 777 本と AAA 212121 本の計 282828 本を飲んだという事実のみを得る.
  • 飲んでいない BBB が 212121 本以上あれば, 飲んでいない BBB 202020 本を減らし, それとは別に AAA 333 本と BBB 212121 本の計 242424 本を飲んだという事実のみを得る.
  • 上記のを少なくとも一方ができる間繰り返す.
  • 繰り返し終了時にもし残りがあれば手順その1のようにする.

ここまでくれば, 簡単な計算に落とし込むことができます.
N=200…0021N=200\dots0021N=200…0021 とすれば, 上記の操作はそれぞれ, N−120\dfrac{N-1}{20}20N−1​ 回行う事ができるため, 飲料を飲んだ事実は, N−120∗28+N−120∗24\frac{N-1}{20}*28+\frac{N-1}{20}*2420N−1​∗28+20N−1​∗24 本分得ます.
この操作を終えた時, A,BA,BA,B はそれぞれ 111 本ずつ残るので, (結局先ほど懸念した中途半端に残ったら手順1をしようと言っていたのをするまでもなく, ) これらを飲んだ本数に加えれば, 公式解説の

M=N−120∗(24+28)+2M=\frac{N-1}{20}*(24+28)+2M=20N−1​∗(24+28)+2

という式を得ます.

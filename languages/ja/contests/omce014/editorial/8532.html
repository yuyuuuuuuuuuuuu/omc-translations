問題文は「自己ループと多重辺を持たない NNN 頂点 N+1N+1N+1 辺の無向グラフであって，各頂点の次数が 111 以上かつ移動を NNN 回繰り返すことが出来るものの数を求めよ」というように言い換えられる．そこで，まず以下の補題を確認しておく．

補題． 任意の頂点について，その次数は 222 以上である．

証明． まず，複数の猫が同じ都市に存在することが出来ないことから，移動の後は常に，各都市に 111 匹ずつ猫がいることが分かる．ここで，次数が 111 の頂点があったと仮定し， 次数が 111 の頂点に注目すると，移動に際しその頂点から出発する猫と，その頂点に到達する猫が両方存在することになる．このとき，それらの猫は共に次数 111 の頂点につながったただ一つの辺を通ることになり，猫がすれ違えないことに矛盾する．よって，背理法により示された．

　ここで，各辺は 222 つの頂点を結んでいることから，全頂点の次数の和は 2(N+1)=2N+22(N+1) = 2N + 22(N+1)=2N+2 と分かる．これと補題の結果を合わせると，頂点の次数の組み合わせは

  • 444 となるものが 111 個，222 となるものが N−1N-1N−1 個
  • 333 となるものが 222 個，222 となるものが N−2N-2N−2 個

のいずれかとなることが分かる．まず，前者の場合を考えると，次数が 444 の頂点以外は全て次数が 222 となっていることから，次数が 444 の頂点を通る閉路が丁度 222 つ存在することが分かる．そこで，それらを C1，C2C_1，C_2C1​，C2​ とする．また，次数が 444 の頂点を AAA とする．まず，AAA にいる猫が C1C_1C1​ を構成する頂点に移動する場合を考える．すると，C1C_1C1​ を構成する頂点にいる猫は，いずれも C1C_1C1​ に沿って時計回り，あるいは反時計回りに 111 つずつ移動するしかない．しかし，この時，AAA と辺で結ばれた C2C_2C2​ の頂点について，そこに到達する猫がいないものが存在することになり，条件を満たすように移動できないと分かる．C2C_2C2​ を構成する頂点に移動する場合も同様であるため，この場合は条件を満たすグラフは存在しないと分かる．
　次に，後者の場合を考える．次数 333 の頂点をそれぞれ AAA と BBB とする．まず，AAA と BBB が直接辺で結ばれていない場合を考える．この時，AAA と辺で結ばれた 333 頂点はいずれも次数が 222 となっている．そこで，それらの 333 頂点を X,Y,ZX, Y, ZX,Y,Z とする．この時，AAA にいた猫が XXX に，YYY にいた猫が AAA に移動したとして一般性を失わないため，そうする．すると，ZZZ にいた猫は，ZZZ と辺で結ばれた AAA 以外の頂点に移動することになり，このような頂点は 111 つしかないことから，ZZZ に到達する猫が存在しないことになり，条件を満たすように移動できないと分かる．よって，AAA と BBB は直接辺で結ばれていなくてはならないと分かる．また，このとき，AAA と BBB を結ぶ辺を 111 本取り除くことによって，全ての頂点の次数が 222 となり，その結果全ての連結成分が閉路となっていることが分かる．この場合，全ての猫が自身のいる頂点が属する閉路に沿って反時計回りに 111 ずつ移動することを繰り返せば条件を満たすため，辺を取り除く前のグラフについても条件を満たすことが分かる．よって，このようなグラフの数を数えればよい．

　A∪B∪C∪DA \cup B \cup C \cup DA∪B∪C∪D が 111 を含むことより, A,B,C,DA,B,C,DA,B,C,D のうち少なくとも 111 つは 111 を含み, 逆に A∩B∩C∩DA \cap B \cap C \cap DA∩B∩C∩D が 111 を含まないことより, A,B,C,DA,B,C,DA,B,C,D のうち少なくとも 111 つは 111 を含まない. したがって, 111 がどの集合に含まれるかは 141414 通り考えられ, 他の要素についても同様に考えることで, 解答は 144=3841614^4=\textbf{38416}144=38416 である.

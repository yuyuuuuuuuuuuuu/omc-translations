a+ba+ba+b と ababab の一の位が等しいとき，少なくとも a+ba+ba+b と ababab の偶奇は一致することが必要です．
　 aaa と bbb の偶奇で場合分けすると，

  • aaa が奇数， bbb が奇数のとき， a+ba+ba+b は偶数， ababab は奇数となり偶奇は一致しない．
  • aaa が奇数， bbb が偶数のとき， a+ba+ba+b は奇数， ababab は偶数となり偶奇は一致しない．
  • aaa が偶数， bbb が奇数のとき， a+ba+ba+b は奇数， ababab は偶数となり偶奇は一致しない．
  • aaa が偶数， bbb が偶数のとき， a+ba+ba+b は偶数， ababab は偶数となり偶奇は一致する．

　したがって a+ba+ba+b と ababab の偶奇が一致することの必要十分条件は aaa と bbb が両方偶数であることです．
　なのでこの問題において， a,ba,ba,b が両方 111 以上 999 以下の偶数である場合（ 161616 通り）をすべて試すと答えを得ます．

In the xyzxyzxyz space, we consider a path from point (0,0,0)(0,0,0)(0,0,0) to point (1000,1000,1000)(1000,1000,1000)(1000,1000,1000) by repeatedly using one of the following movements. How many such paths exist where the number of times a different movement method was used immediately prior is exactly 444?

  • Movement AAA: From (x,y,z)(x,y,z)(x,y,z) to (x+1,y+1,z)(x+1,y+1,z)(x+1,y+1,z).
  • Movement BBB: From (x,y,z)(x,y,z)(x,y,z) to (x,y+1,z+1)(x,y+1,z+1)(x,y+1,z+1).
  • Movement CCC: From (x,y,z)(x,y,z)(x,y,z) to (x+1,y,z+1)(x+1,y,z+1)(x+1,y,z+1).

For example, when moving as AAABBCAABBBAAABBCAABBBAAABBCAABBB, the number of times a different movement method was used immediately prior is exactly 444.
